# Programmatic Image Editor

An AI agent that generates images by writing code. From [On Programmatic Tool Calling](https://www.guillemus.com/on-programmatic-tool-calling).

## Setup

```bash
git clone https://github.com/guillemus/programatic-tool-calling
cd programatic-tool-calling
bun install
```

## Usage

```bash
export OPENAI_API_KEY=your-key-here
bun src/index.ts
```

Output images are saved to `./data/`.
